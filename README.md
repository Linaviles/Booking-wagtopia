# Booking-wagtopia

# Description

    Simple booking website that is mobile-friendly! Seed data provided for groomers Alice and Bob.

# Prerequisites

    Node.js 18+, npm

API base URL http://localhost:3000

# Endpoints

    GET /api/bookings, POST /api/bookings, backend/server.js


# Backend Set Up 

    cd backend
    npm install
    node server.js

# Frontend Set Up

    Option 1:
    Open in browser frontend/bookings.html
    
    Option 2:
    cd frontend
    npx http-server or python3 -m http.server
    visit http://localhost:8080/bookings.html
